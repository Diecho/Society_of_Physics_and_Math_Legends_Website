# Society of Physics and Math Legends Website

Website for the Society of Physics and Math Legends (SPML) at Clovis Community College.

## Overview
This website was developed to establish an online presence for the club, provide information to students, and support outreach efforts for recruitment and events.

## Features
- Information about the club and its mission
- Event announcements and updates
- Resources for students interested in physics and mathematics
- Clean and responsive design for accessibility across devices

## Technologies Used
- HTML
- CSS
- JavaScript

## My Role
- Designed and developed the website
- Worked with the club to define requirements and structure
- Implemented layout, styling, and content organization
- Deployed and maintained the live site
